# Async Task API

## 説明

[Nest](https://github.com/nestjs/nest)で実装した非同期タスクAPI。

## セットアップ

```bash
$ npm install
```

## 実行方法

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## テスト

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
